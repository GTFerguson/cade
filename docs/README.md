# CADE Documentation

Welcome to the CADE documentation. This is the main navigation hub for all project documentation.

## What is CADE?

**CADE** (Claude Agentic Development Environment) is an agent-first development environment with Claude Code in a terminal shell as its centerpiece. The interface provides a unified workspace where AI-assisted development is the primary workflow.

### Access Anywhere, Any Device

CADE's web-based architecture enables flexible deployment and access:

- **Desktop App** — Native Tauri application with bundled backend for local development
- **Web Browser** — Deploy on a remote server (EC2, VPS) and access from any browser
- **Mobile** — Responsive mobile interface with touch toolbar and file explorer for phones and tablets
- **Multi-Device** — Connect to the same session from multiple devices simultaneously

This means you can start work on your desktop, check progress from your phone, and continue on a different machine — all on the same persistent session.
