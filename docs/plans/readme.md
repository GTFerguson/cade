# Plans

Active and completed plans for CADE. See subdirectories for organized content.

## Directory Structure

| Directory | Contents |
|---|---|
| [[handoff/readme\|handoff]] | Implementation notes from in-flight sessions |
| [[planning/readme\|planning]] | Feature proposals and planning docs |
| [[shipped/readme\|shipped]] | Completed plans that have graduated to architecture docs |
| [[design/readme\|design]] | HTML mockups and visual explorations |

## Lifecycle

1. **Planning** — proposals and roadmaps live in `planning/`
2. **Handoff** — working implementation notes go in `handoff/`
3. **Shipped** — completed plans move to `shipped/` before the plan doc is deleted and replaced by an architecture doc
