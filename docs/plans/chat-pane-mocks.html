The quadratic formula solves any equation of the form ax² + bx + c = 0:

x = −b ± √b² − 4ac 2a

Where the discriminant Δ = b² − 4ac determines the nature of the roots:

DiscriminantRoots
Δ > 0Two distinct real roots
Δ = 0One repeated real root
Δ < 0Two complex conjugate roots

Python implementation:

pythonimport cmath

def quadratic(a: float, b: float, c: float) -> tuple:
    """Solve ax² + bx + c = 0."""
    discriminant = b**2 - 4*a*c
    sqrt_d = cmath.sqrt(discriminant)
    x1 = (-b + sqrt_d) / (2 * a)
    x2 = (-b - sqrt_d) / (2 * a)
    return x1, x2

Design Decisions

Input Style

Terminal prompt (❯ + underline field) instead of boxed textarea. Matches auth input and file tree search patterns already in CADE.

No Bubbles

Bubbles with border-radius violate the zero-radius terminal aesthetic. Full-width messages with left-border indicators instead.

Statusline

Vim-style statusline at bottom shows mode, provider, and token count. Matches the existing viewer statusline pattern exactly.

Math Blocks

KaTeX renders with a purple left-border indicator (distinguishing from blue blockquotes). Centered alignment for display math.

Mermaid Blocks

Bordered container with a mermaid language tag (same pattern as code blocks). Dark background preserves the theme.

Streaming

Block cursor () in accent-blue. Blinks at 1Hz. Disappears when response completes.

Thinking

Collapsible block with muted left-border. Shows duration. Collapsed by default after completion, open during streaming.

Role Colors

You (user) — green (matches edit mode)
Assistant — blue (matches Claude indicator)
System — muted (informational)
Error — red (matches error states)
